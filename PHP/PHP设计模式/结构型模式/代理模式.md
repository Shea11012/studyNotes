## 代理模式（proxy）

是为其他对象提供一种代理来控制该对象的访问。通常代理类和被代理的类拥有相同的功能，它们也通常实现相同的接口或者继承自同一个父类。

```php
// 定义老板类
class Boss
{
    public function show()
    {
        echo 'Boss已查看该文件';
    }
}

// 定义秘书类
class Secretary extends Boss
{
    public function show()
    {
        echo '转交给boss';
        parent::show();
    }
}

$t = new Secretary();
$t->show();
```
