### NERDTree（目录树） 常用快捷键

| 命令     | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| h j k l  | 移动光标定位                                                 |
| ctrl+w+w | 光标在左右窗口切换                                           |
| ctrl+w+r | 切换当前窗口左右布局                                         |
| ctrl+p   | 模糊搜索文件                                                 |
| gT       | 切换到前一个 tab                                             |
| g t      | 切换到后一个 tab                                             |
| o        | 打开关闭文件或者目录，如果是文件的话，光标出现在打开的文件中 |
| O        | 打开节点下的所有目录                                         |
| X        | 合拢当前节点的所有目录                                       |
| x        | 合拢当前节点的父目录                                         |
| i 和 s   | 水平分割或纵向分割窗口打开文件                               |
| u        | 打开上层目录                                                 |
| t        | 在标签页中打开                                               |
| T        | 在后台标签页中打开                                           |
| p        | 到上层目录                                                   |
| P        | 到根目录                                                     |
| K        | 到同目录第一个节点                                           |
| J        | 到同目录最后                                                 |
| m        | 显示文件系统菜单（添加、删除、移动操作）                     |
| q        | 关闭                                                         |
