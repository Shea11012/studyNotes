|   ctrl+shift+N    | shift + command + o   |               查找文件               |

| ctrl+shift+alt+T  |                       |             代码片段重构             |
|       alt+j       | control + G           |                 选中                 |
|    alt+shift+j    | control + shift + G   |               撤销选中               |
| alt+ctrl+shift+j  | control + command + G |               全部选中               |
