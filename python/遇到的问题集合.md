# 遇到的问题集合
list = [1,2,3,4]    
''.join(list)   
将上面的list转成str会报一个错误：expected str instance, int found   
解决办法：  
因为list里有整形，所以先将list里的整形转换用str()  
''.join(str(list))
      
在迭代器中不能直接删除列表元素，需要把原列表拷给迭代器，再对原列表进行删除  
```python
for each_fish in fish[:]:
    if each_fish.move() == pos:
        tortoise.eat()
        fish.remove(each_fish)
        print('有一条鱼被吃掉了')
```
